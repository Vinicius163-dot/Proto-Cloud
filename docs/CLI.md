# CLI UX - proto-cloud (MVP)

Comandos iniciais:
- `proto-cloud version`
- `proto-cloud regions list`
- `proto-cloud instances list --region <region>`

Exemplo:
- `proto-cloud regions list`
  - saída: lista de regiões (ex: us-east-1, eu-west-1)
- `proto-cloud instances list --region us-east-1`
  - saída: lista com ID, type, state, name
